My Little Shop - React Project 
​This project is a functional e-commerce web application built entirely using React.js. It demonstrates a robust implementation of global state management and a component-based architecture for a seamless shopping experience.
​
. Key Features
​Context API Management: Centralized shopping cart logic for adding and removing items.
​Dynamic Routing: Multi-page navigation between the Shop catalog and the User Cart.
​Real-time State Updates: Instant UI feedback for p
product quantities and cart status.
Structured Data: Dynamic rendering of products from a dedicated data source.

. Tech Stack
Library: React.js
Routing: React Router Dom
Styling: Bootstrap / CSS
Icons: FontAwesome

. Installation & Setup
To run this project locally, follow these steps:


1- Clone the repository:
git clone https://github.com/my-negar/react-shop-project.git

2- Install dependencies:
npm install

3- Start the development server:
npm start
